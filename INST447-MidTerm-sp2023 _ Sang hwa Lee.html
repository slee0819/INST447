(12) Explain why the mean different after you drop all the missing rows from the dataframe. What are our options for dealing with these missing values?

The reason for the difference in means is that the size of the dataset decreases while deleting missing rows. Deleted rows originally affected the overall mean, but after they are deleted, they are no longer influential. This results in a difference in the mean.

There are many ways to deal with missing values:

1.Delete missing values.

2.Replace missing values (ex. replace missing values with mean, median, or least frequent values in the column).

3.Fill missing values with specific constants.

(22) Create a pie chart that shows the gender of the voters for those 5 years with matplotlib.

The syntact for the pie function is:

plt.pie(list_of_values, labels=list_of_labels, autopct='%1.1f%%', startangle=90)
Values you need to change:

  • list_of_values is a list of the values that define the slices of the pie.
  • list_of_labels is a list of the values that you want to use as labels for the slices.

Values you don't need to change

  • autopct adds the percentages text to the slices.
  • startangle rotated the pie so that it is left/right oriented instead of top/bottom.

# Define the age groups columns
